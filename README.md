<h1>Selenium course project module at stepik.org</h1>

<p>В данном репозитории ведется финальный проект курса <a href='https://stepik.org/course/575/syllabus'> Автоматизация тестирования с помощью Selenium и Python </a>

<p>base_page.py - тут мы храним методы, которые применяются по всему проекту вообще, всё завернуто в класс, чтобы было удобно импортировать.

<p>locators.py - тут мы храним локаторы, в виде констант. Локаторы каждой отдельной страницы завёрнуты в класс, чтобы было удобно импортировать.

<p>main_page.py - тут мы храним методы по конкретной странице, завернутые в класс этой странице. Класс этот - условный MainPage - наследник класса BasePage, чтобы можно было пользоваться методами, описанными в base_page.py.
